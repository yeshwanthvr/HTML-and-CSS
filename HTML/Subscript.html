Use of the subscript tag to write chemical formulas


Hello everyone, today I am going to show you some chemical formulas and equations.


1) N2 + H2 -> NH3
2) CuSO4 + NaOH -> Na2SO4 + Cu(OH)2
3) NH3 + Hcl -> NH4cl
4) CaCO3 -> CaO + CO2
5) NaOH + Hcl -> Nacl + H2O



The numbers highlighted in crimson colour are written with the help of the subscript tag.
