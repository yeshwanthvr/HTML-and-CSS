Use the information from the picture given below to find the value of the following :-
  1. sin A
  2. cos A
  3. tan A
  4. cot A
  5. sec A
  6. cosec A


Answer :-

Given side AC = hypotenuse , side AB = base and side BC = perpendicular

1) ∵ sin A = perpendicular / hypotenuse
∴ sin A = 3 / 5

2) ∵ cos A = base / hypotenuse
∴ cos A = 4 / 5

3) ∵ tan A = perpendicular / base
∴ tan A = 3 / 4

4) ∵ cot A = base / perpendicular
∴ cot A = 4 / 3

5) ∵ sec A = hypotenuse / base
∴ sec A = 5 / 4

6) ∵ cosec A = perpendicular / hypotenuse
∴ cosec A = 5 / 3
